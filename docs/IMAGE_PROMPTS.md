## 提示词使用建议

1. **分辨率参数**：添加 `--ar 16:9` 或 `--ar 4:5` 根据用途调整
2. **风格参数**：可添加 `--style raw` 获得更真实的渲染
3. **质量参数**：使用 `--q 2` 或 `--quality 2` 提高细节
4. **负面提示词**：`--no text, watermark, logo overlay, distorted face, unrealistic lighting`
