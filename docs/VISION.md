## 一、产品定位

### 1.1 一句话定义

**Agentic Mirror 是一款集智能追踪、健康监测、AI 美妆推荐和电商于一体的智能美妆镜，让每位女性都能轻松找到最适合自己的美丽方案。**

### 1.2 核心价值主张

| 用户痛点 | Agentic Mirror 解决方案 |
|----------|------------------------|
| 不知道什么妆容适合自己 | AI 分析面部特征，推荐匹配妆容 |
| 跟不上美妆潮流 | 实时抓取小红书/抖音热门妆容 |
| 化妆技术有限 | Step-by-step 视频教程引导 |
| 产品选择困难 | 智能推荐最适合的产品 |
| 皮肤状态不稳定 | 持续监测，动态调整推荐 |
| 生理期/特殊日子妆容需求 | 结合日历智能适配 |

### 2.2 用户场景

#### 场景一：职场白领 Amy
- **背景**: 28岁，互联网公司产品经理
- **痛点**: 每天早上化妆时间紧张，经常不知道画什么妆
- **使用场景**:
  - 早上起床，Mirror 自动开机，分析今日皮肤状态
  - 同步日历发现今天有重要会议
  - 推荐 "干练职场妆"，并提供5分钟快速版教程
  - 检测到口红快用完，推荐补货链接

#### 场景二：大学生小美
- **背景**: 20岁，大三学生
- **痛点**: 想尝试新妆容但怕翻车
- **使用场景**:
  - 周末约会前，查看小红书热门约会妆
  - Mirror 分析她的脸型，推荐3款适合的改良版
  - 选择后跟着教程一步步化妆
  - 完成后 Mirror 拍照记录，下次可一键复刻

#### 场景三：新手妈妈 Linda
- **背景**: 32岁，全职妈妈
- **痛点**: 产后皮肤状态差，没时间研究护肤
- **使用场景**:
  - Mirror 监测到皮肤干燥、有色斑
  - 推荐针对性护肤流程和产品
  - 结合睡眠数据（手表同步），调整护肤建议
  - 每周生成皮肤改善报告

## 四、差异化竞争优势

### 4.1 竞品分析

| 竞品 | 优势 | 劣势 | Agentic Mirror 差异化 |
|------|------|------|----------------------|
| HiMirror | 皮肤分析精准 | 无妆容推荐 | 完整美妆闭环 |
| 魔镜 | 价格便宜 | 功能单一 | AI + 电商 |
| 玫琳凯镜 | 品牌背书 | 仅自家产品 | 开放平台，全品类 |
| 美图魔镜 | AR 试妆 | 无硬件 | 软硬结合 |

### 4.2 核心壁垒

1. **软硬件一体化**: 自研云台 + AI 算法，体验最优
2. **数据飞轮**: 用户越多，推荐越准，形成网络效应
3. **Agentic Commerce**: 不只是推荐，是主动服务
4. **内容生态**: UGC 妆容分享，社区运营
